# Smart_AgroAI-Bridge
Smart AgroAI Bridge is an AI-powered agricultural assistant designed for rural African farmers.

The system uses computer vision and conversational AI to detect crop diseases, provide agricultural recommendations, and interact in Tunisian dialect.

## Features
- Plant disease detection
- AI agricultural assistant
- Tunisian dialect interaction
- Voice support
- Scientific agricultural knowledge integration

## Technologies
- Computer Vision (CNN)
- Whisper
- RAG
- Edge TTS
- Python

## Goal
Improve agricultural productivity and reduce crop losses through accessible AI technologies adapted to African rural environments.

## Future Vision
- Offline support
- Satellite integration
- Smart irrigation prediction
- Multi-crop disease detection
